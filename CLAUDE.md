# 儿童英语语法小手册 — 项目准则

面向中国小学生(难度按**三年级**,但**文案不出现年级字样**)的英语语法入门手册。纯静态 HTML,部署 GitHub Pages,必须**手机 / iPad 友好**。

## 最高准则:对标美国本土权威体系
语法体系一律以**美国头部私立 / 古典(classical)/ 资优小学**为准
(参考 First Language Lessons、Rod & Staff、Michael Clay Thompson、Shurley English、Reed-Kellogg 句子图解、Warriner's)。
- 句子分析地基 = **Subject + Predicate(主语 + 谓语)**;不"主谓宾"先行,**不教中国式"五大句型(SV/SVO/SVC…)"**。
- be 动词句 = 主语 + **系动词(linking verb)** + **表语(subject complement)** = 主系表(**不是**"主谓补")。
- 任何术语 / 编排取舍,先问:"古典 / 头部私立小学怎么教这个?"

## 教学法原则
1. **一关一个核心概念**;地基用"最小的真命题",绝不教半真半假。
2. **先建立语言感觉(language intuition),再总结规则**(归纳式):先用真实例子 / 问答 / 情景让孩子*感觉*到怎么用,规律只在后面做一句小结——不从规则、术语、词形表讲起,不要求背条文。语感在前,规则在后。
3. **由整到细、渐进拆分**:先 `主语 | 谓语`,再往谓语里拆 动词 / 宾语 / 系动词 / 表语。
4. **文案精简、低阅读压力**:定义一句话讲完,靠图和例句说话;术语配口语化解释,别堆长段。
5. 中英双语,语气童趣、鼓励、**陪伴式**——**绝不说教**:不用"一定要记住 / 必须 / 要牢记 / 不用纠结"这类命令、规训、叮嘱口吻;靠发现、比喻、邀请,让孩子自己感受。

## 设计语言
- 手账 / 贴纸风;**两层语法(MCT 式),用两个通道分别承载**:
  - 🎨 **颜色 = 成分**(角色:主语/谓语/宾语/系动词/表语…),跨关一致、随概念**渐进**加色。
  - 🔊 **点击朗读**:独立词汇(封面/列举)念**词类**(名词/动词…);**进了句子的核心词念「词类+角色」**(名词作主语、名词作宾语、代词作宾语…);第 1 关只念**成分**(主语/谓语)。修饰词(形/副词等)、形态展示(时态/比较级)只念词类。
  - 口诀:看颜色 + 听声音,知道它**是什么词、在当什么角色**。朗读用 Web Speech 中文 TTS,读中文术语、不读英文单词。

## 工程与发布
- **每个栏目整体隔离在自己的目录**(语法手册 `grammar/`、KET 词汇默写 `ket/`);根目录只放站点级文件(栏目导航页 `index.html`、`404.html`、`CNAME` 等),栏目之间不互相引用。
- 每关一个 `grammar/lesson-NN.html`,共享 `grammar/assets/style.css` + `grammar/assets/app.js`,**全部相对路径**(目录内互相引用)。
- `grammar/index.html` = 封面 + 11 关目录(未完成的标 `soon`);根 `index.html` = 栏目导航,新栏目上线时在这里加卡片。
- `ket/` = KET 核心词汇 A4 默写卷(词表 CSV → 生成脚本 → HTML/PDF,目录页 `ket/index.html`;工作流 → `/ket` skill)。
- 分支 **master**;自定义域名 `english.hi-ruby.com`(根目录 `CNAME`)。本地预览 `make up`。
- **重大改动 / 体系取舍先和用户讨论**;改完用 playwright 截图自检(手机 + iPad 视口);确认后 **commit + push 到 master**(用户已授权直接 push;commit 末尾加 Co-Authored-By 行)。

## 11 关大纲
1 句子(主语+谓语) · 2 名词+冠词(a/an/the,→引入宾语) · 3 代词 · 4 动词(动作/系动词,→引入系动词+表语/主系表) · 5 时态 · 6 形容词 · 7 副词 · 8 比较级 · 9 介词 · 10 连词 · 11 句子四类+标点。
(成分细分就近下沉:宾语并入名词关、系动词+表语并入动词关;颜色随成分出现**渐进**加。词性关里点击色块念词性,成分仍用既定颜色。)
